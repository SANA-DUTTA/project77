Visual Studio Code is a code editor made by Microsoft for Windows, Linux and macOS.[9] Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add additional functionality. Microsoft has released most of Visual Studio Code's source code on GitHub


In computer technology, a console can mean different things depending on the context. It usually means a combination of a display monitor and an input device, usually a keyboard and mouse pair, which allows a user to input commands and receive visual output from a computer or computer system. It can also refer to specialized computers only meant to play video games. A console is also known as a computer console, a system console or simply as a terminal.
